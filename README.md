## Getting Started

To run this project locally, follow these steps:

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/nuraskhan/week-14
2. **Go inside directory:**

   ```bash
   cd Nuraskhan
3. **Install the npm:**

   ```bash
   npm install
4. **Run the code:**

   ```bash
   npm run dev
5. **Install the storybook:**

   ```bash
   npx storybook@latest init
6. **Run the storybook:**

   ```bash
   npm run storybook
